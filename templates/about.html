About Student Management System

📚 Project Overview

Student Management System is a web application developed using Flask and MySQL to manage student records efficiently. It allows administrators to add, edit, delete, search and generate reports for students through a simple and user-friendly interface.

✨ Main Features

  • ✔ Dashboard with statistics
  • ✔ Add Student
  • ✔ Edit Student Details
  • ✔ Delete Student
  • ✔ Search Students
  • ✔ Department-wise Student List
  • ✔ Reports Dashboard
  • ✔ Student Report
  • ✔ Department Report
  • ✔ New Admissions Report
  • ✔ CSV Export

🛠 Technologies Used

Frontend HTML, CSS, JavaScript
Backend Python Flask
Database MySQL
Charts Chart.js
