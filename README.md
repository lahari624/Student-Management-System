# 🎓 Student Management System

A web-based **Student Management System** developed using **Python Flask** and **MySQL** to efficiently manage student records. The application provides an interactive dashboard, student management, department management, reports, search, pagination, and data visualization through charts.

---

## 📌 Features

- 🔐 Secure Admin Login Authentication
- 📊 Interactive Dashboard
- 👨‍🎓 Student Management (CRUD Operations)
- 🔍 Search Students
- 📄 Student Profile Page
- 🏫 Department Management
- 📈 Dashboard Charts
  - Department Distribution
  - Gender Distribution
  - Fee Status Distribution
- 📑 Reports Module
  - Student Report
  - Department Report
  - New Admissions Report
- 📤 Export Student Data to CSV
- 📄 Pagination
- 📱 Responsive User Interface
- 🎨 Modern Admin Dashboard Design

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/lahari624/Student-Management-System.git
```

### Navigate to Project Folder

```bash
cd Student-Management-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure MySQL

1. Create a MySQL database.
2. Import the required tables.
3. Update the database credentials in `database.py`.

### Run the Project

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
